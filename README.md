# Todo List

React's Todo List

This todo list is build on reactjs along with the tailwindcss utility first CSS framework, which provides the user to add, update and delete task.
## Features

- Add
- Update
- Delete


## Tech Stack

- React
- Javascript
- Tailwindcss
